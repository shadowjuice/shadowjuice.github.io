第六天

今天学了很多东西，早上开始学习py教程里的模块的东西，安装了pycharm还有anaconda，配置了一下py的环境，把常用的库都导入进去了，顺带学了一下爬虫的原理还有思路，看了一点requests的知识，然后上网学习了一下，把web的第一道题做了。
思路就是首先进行网页检查，把网页的内容存储下来，然后提取出数学表达式，利用函数计算出数值，再找到网页提交的名称，把计算出的结果赋值给name，因为网页采用的是get方法，学习一下post的方法，最后在提交，然后输出一下，就可以发现结果中带有flag。

import requests

url = ‘http://123.207.149.64:23331/calculator/'

a = requests.session()

a1 = a.get(url)

b = a1.text

b1 = b.find(‘“exp”>’)

b2 = b.find(‘‘ , b1)

b3 = b [b1+6:b2-2]

b4 = eval(b3)

payload = {‘answer’:b4}

a2 = a.get( url, params=payload)

print(a2.text)
